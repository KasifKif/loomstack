## Per-task workflow

  1. `git checkout main && git pull origin main`
  2. `git checkout -b feat/<task-slug>`
  3. `cd blueprint && uv sync --all-extras`
  4. Implement, verify: ruff check, ruff format, mypy, pytest
  5. Commit, push, open PR against main
  6. Tell user to merge, then start next task
